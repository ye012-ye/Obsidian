# 分页、排序、高亮与聚合

> [!tip] 本章目标
> 你要能把搜索页面做完整：查得到、翻得动、排得准、看得清、还能统计筛选项。

## 普通分页

```java
int page = 1;
int size = 20;

client.search(s -> s
        .index("products_v1")
        .from((page - 1) * size)
        .size(size)
        .query(q -> q.match(m -> m.field("name").query("Java")))
        , ProductDoc.class);
```

> [!warning] 深分页问题
> `from + size` 翻到很深时，ES 需要跳过大量结果，成本会快速变高。普通后台列表可以用，面向用户的无限滚动要考虑 `search_after`。

## 排序

```java
client.search(s -> s
        .index("products_v1")
        .query(q -> q.match(m -> m.field("name").query("Java")))
        .sort(sort -> sort.field(f -> f.field("price").order(co.elastic.clients.elasticsearch._types.SortOrder.Asc)))
        .sort(sort -> sort.field(f -> f.field("createdAt").order(co.elastic.clients.elasticsearch._types.SortOrder.Desc)))
        , ProductDoc.class);
```

排序字段通常应是 `keyword`、数值、日期字段，不要直接拿 `text` 排。

## 高亮

```java
var response = client.search(s -> s
        .index("products_v1")
        .query(q -> q.match(m -> m.field("name").query("Java")))
        .highlight(h -> h
                .fields("name", f -> f
                        .preTags("<em>")
                        .postTags("</em>")
                )
        ), ProductDoc.class);

response.hits().hits().forEach(hit -> {
    var highlight = hit.highlight().get("name");
    System.out.println(highlight);
});
```

> [!success] 高亮使用建议
> 后端返回高亮片段，前端负责安全渲染。不要让用户输入直接变成不受控 HTML。

## 聚合：按分类统计

```java
var response = client.search(s -> s
        .index("products_v1")
        .size(0)
        .aggregations("category_count", a -> a
                .terms(t -> t.field("category"))
        ), Void.class);

var buckets = response.aggregations()
        .get("category_count")
        .sterms()
        .buckets()
        .array();

buckets.forEach(b -> System.out.println(b.key().stringValue() + " -> " + b.docCount()));
```

官方 Java Client 聚合示例也建议：只做统计、不关心命中文档时，把 `size` 设置为 0，并把搜索结果类型设成 `Void`。

## 聚合：价格区间

```java
client.search(s -> s
        .index("products_v1")
        .size(0)
        .aggregations("price_ranges", a -> a
                .range(r -> r
                        .field("price")
                        .ranges(rr -> rr.to("50"))
                        .ranges(rr -> rr.from("50").to("100"))
                        .ranges(rr -> rr.from("100"))
                )
        ), Void.class);
```

## 本章小结

> [!info] 搜索页四件套
> 查询负责召回，排序负责顺序，高亮负责可读性，聚合负责筛选导航。
