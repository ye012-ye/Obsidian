# 13 · 拓扑排序与 DAG

> 有向图里，只要题目提到"先做 A 再做 B"、"依赖关系"、"课程先修"，就要想到拓扑排序。

---

## 1. 什么是拓扑排序

对于一个 **有向无环图（DAG）**，把所有点排成一个序列，使得：

> 每条有向边 `u -> v` 中，u 都出现在 v 前面。

```
1 -> 3
2 -> 3
3 -> 4

合法拓扑序：
1, 2, 3, 4
2, 1, 3, 4
```

只要有环，就不存在拓扑序。

---

## 2. 入度的概念

**入度**：有多少条边指向这个点。

```
1 -> 3 <- 2
     |
     v
     4

in[1] = 0
in[2] = 0
in[3] = 2
in[4] = 1
```

拓扑排序的关键观察：
> 入度为 0 的点，没有任何前置条件，可以先做。

---

## 3. Kahn 算法（队列版拓扑排序）

步骤：
1. 把所有入度为 0 的点入队
2. 每次出队一个点 u，放进答案
3. 删除 u 的所有出边，让邻居入度减 1
4. 若某个邻居入度变成 0，就入队

---

## 4. 图解

```
1 -> 3 -> 5
2 -> 3
2 -> 4 -> 5

初始入度：
in[1]=0, in[2]=0, in[3]=2, in[4]=1, in[5]=2

队列初始：[1, 2]

弹 1：
拓扑序 = [1]
3 的入度 2 -> 1

弹 2：
拓扑序 = [1, 2]
3 的入度 1 -> 0，入队
4 的入度 1 -> 0，入队

队列：[3, 4]

弹 3：
5 的入度 2 -> 1

弹 4：
5 的入度 1 -> 0，入队

弹 5：
结束
```

## 5. 模板

```java
List<Integer> topoSort(int n, List<List<Integer>> g) {
    int[] in = new int[n + 1];
    for (int u = 1; u <= n; u++)
        for (int v : g.get(u)) in[v]++;

    ArrayDeque<Integer> q = new ArrayDeque<>();
    for (int i = 1; i <= n; i++)
        if (in[i] == 0) q.offer(i);

    List<Integer> topo = new ArrayList<>();
    while (!q.isEmpty()) {
        int u = q.poll();
        topo.add(u);
        for (int v : g.get(u)) {
            if (--in[v] == 0) q.offer(v);
        }
    }
    return topo;
}
```

若 `topo.size() < n`，说明图里有环。

---

## 6. 拓扑排序最经典的两个用途

### 6.1 判环

如果最后没排完所有点，说明存在环。

这就是：
- 课程表（LC 207）
- 任务依赖合法性判断

### 6.2 DAG 上做 DP

因为拓扑序保证了"前面的状态已经算完"，所以特别适合做 DP。

---

## 7. DAG 上的最长路 / 路径计数

### 7.1 最长路

在一般图里最长路很难，但在 DAG 里可以按拓扑序转移：

```java
for (int u : topo) {
    for (int v : g.get(u)) {
        dp[v] = Math.max(dp[v], dp[u] + 1);
    }
}
```

### 7.2 路径计数

`dp[u] = 到 u 的方案数`

```java
dp[s] = 1;
for (int u : topo) {
    for (int v : g.get(u)) {
        dp[v] += dp[u];
    }
}
```

图上理解：
```
1 -> 3 -> 5
2 -> 3

到 3 的方案数 = 到 1 的方案数 + 到 2 的方案数
```

## 8. 什么时候想到拓扑排序

关键词：
- 先修关系
- 依赖关系
- 某任务必须在另一个前完成
- 是否存在循环依赖
- DAG 上最优解 / 方案数

---

## 9. 踩坑提醒 ⚠️

1. **拓扑排序只对有向图有意义**。
2. **有环就不存在拓扑序**，别强行套。
3. **可能有多个入度为 0 的点**，所以拓扑序不唯一。
4. **点编号从 0 还是 1**，入度数组别开错。
5. 若题目要求**字典序最小拓扑序**，队列要换成小根堆。
