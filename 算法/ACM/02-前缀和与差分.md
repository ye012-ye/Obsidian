# 02 · 前缀和与差分

> 校赛最高频技巧之一。只要题目出现"**区间和**"、"**区间修改**"，几乎都要用。

---

## 1. 前缀和（Prefix Sum）

### 1.1 要解决什么问题？

**问题**：给数组 `a[1..n]`，查询 `q` 次，每次问区间 `[l, r]` 的和。

**暴力**：每次查询 O(r-l) → 总 O(nq)，n=q=10⁵ 时 **超时**。

**前缀和**：预处理 `S[i] = a[1] + a[2] + ... + a[i]`，查询 O(1)：
> **sum(l..r) = S[r] - S[l-1]**

### 1.2 ASCII 图解

```
下标:   1   2   3   4   5
a[] =   2   5   1   3   4

S[0] = 0
S[1] = 0 + 2 = 2
S[2] = 2 + 5 = 7
S[3] = 7 + 1 = 8
S[4] = 8 + 3 = 11
S[5] = 11+ 4 = 15

S[] =  0   2   7   8  11  15
       └── 下标 0 哨兵 ──┘

查询 sum(2..4) = a[2]+a[3]+a[4] = 5+1+3 = 9
用前缀和 = S[4] - S[1] = 11 - 2 = 9 ✓

可视化（条形图）：
      S[5]=15 ┌─────┐
      S[4]=11 ┌───┐ │
      S[3]= 8 ┌─┐ │ │
      S[2]= 7 ┌┐│ │ │
      S[1]= 2 ┤││ │ │   ← 切掉 S[1]，剩下就是 [2..4] 的和
             0   1 2 3 4 5
```

### 1.3 Java 模板

```java
int n = ...;
int[] a = new int[n + 1];          // 下标从 1 开始方便
long[] S = new long[n + 1];         // 一定用 long！防溢出
for (int i = 1; i <= n; i++) a[i] = ...;
for (int i = 1; i <= n; i++) S[i] = S[i - 1] + a[i];

// 查询 [l, r]
long sum = S[r] - S[l - 1];
```

**踩坑**：如果下标从 0 开始，查询 `[l, r]` 是 `S[r+1] - S[l]`。用哨兵 `S[0]=0` 更不易错。

## 2. 二维前缀和

### 2.1 问题
给 n×m 矩阵 `a`，查询子矩阵 `(r1,c1)~(r2,c2)` 的和。

### 2.2 公式（容斥原理）

```
S[i][j] = 以 (1,1)~(i,j) 为左上角的子矩阵的和
        = S[i-1][j] + S[i][j-1] - S[i-1][j-1] + a[i][j]

查询 (r1,c1)~(r2,c2):
     = S[r2][c2] - S[r1-1][c2] - S[r2][c1-1] + S[r1-1][c1-1]
```

### 2.3 图解

```
要查的矩形 = 整个大矩形 - 上边多出的 - 左边多出的 + 左上角多扣的

┌───────────┐
│   A       │  ← S[r1-1][c2]
├───┬───────┤
│ B │   要  │
│   │  查   │  ← S[r2][c2] - A - B + C
│   │  的   │
└───┴───────┘
    │
    └── C = S[r1-1][c1-1]
```

### 2.4 模板

```java
long[][] S = new long[n + 1][m + 1];
for (int i = 1; i <= n; i++)
    for (int j = 1; j <= m; j++)
        S[i][j] = S[i-1][j] + S[i][j-1] - S[i-1][j-1] + a[i][j];

// 查询
long ans = S[r2][c2] - S[r1-1][c2] - S[r2][c1-1] + S[r1-1][c1-1];
```

## 3. 差分（Difference Array）

### 3.1 要解决什么问题？

**问题**：q 次操作，每次给区间 `[l, r]` 所有元素 **+v**；最后输出数组。

**暴力**：每次操作 O(n) → 总 O(nq)，**超时**。

**差分**：构造 `d[i] = a[i] - a[i-1]`。
- 想给 `[l, r]` 全部 +v → 只要 `d[l] += v, d[r+1] -= v` 两步 O(1)！
- 最后再对 `d` 求一次前缀和就还原出 `a`。

### 3.2 ASCII 图解

```
原数组 a：
i:    1  2  3  4  5
a[]:  0  0  0  0  0

操作 1：[2, 4] 每个 +3  →  d[2] += 3, d[5] -= 3
操作 2：[1, 3] 每个 +1  →  d[1] += 1, d[4] -= 1

d[] 数组变化：
i:    1  2  3  4  5
d[]:  0  0  0  0  0     初始
d[]: +1  0  0 -1  0     op2
d[]: +1 +3  0 -1 -3     op1
      │  │  │  │  │
      ▼  ▼  ▼  ▼  ▼
前缀和恢复 a：
a[1] = 1
a[2] = 1 + 3 = 4
a[3] = 4 + 0 = 4
a[4] = 4 - 1 = 3
a[5] = 3 - 3 = 0

最终 a[] = 1 4 4 3 0   ✓
```

### 3.3 模板

```java
long[] d = new long[n + 2];        // +2 防止 r+1 越界

// 区间加：[l, r] 每个元素 += v
void add(int l, int r, long v) {
    d[l] += v;
    d[r + 1] -= v;
}

// 复原
long[] a = new long[n + 1];
for (int i = 1; i <= n; i++) a[i] = a[i - 1] + d[i];
```

## 4. 前缀和 + 差分的 "对偶"

```
            前缀和 S         差分 d
操作        单点改 → 区间问   区间改 → 单点问
构造        S[i]=S[i-1]+a[i]  d[i]=a[i]-a[i-1]
还原        a[i]=S[i]-S[i-1]  a[i]=d[1]+...+d[i]
```

**一句话**：前缀和是差分的"反函数"。

## 6. 扩展进阶：前缀和 + HashMap 统计"子数组个数"

前缀和不只用来"求区间和"，还常用来统计：
- 和恰好等于 k 的子数组个数
- 0/1 数组里 0 和 1 数量相同的子数组个数
- 前缀和模某个数相同的区间个数

### 经典例：和为 k 的子数组（LC 560）

若 `pre[j] - pre[i] = k`，等价于 `pre[i] = pre[j] - k`。
所以扫到当前位置 j 时，只要看前面出现过多少次 `pre[j] - k`。

```
a = [1, 2, 1, 2], k = 3
pre: 0 1 3 4 6

扫到 pre=3：需要 pre-k=0，出现 1 次 → 新增 1 段 [1,2]
扫到 pre=4：需要 pre-k=1，出现 1 次 → 新增 1 段 [2,1]
扫到 pre=6：需要 pre-k=3，出现 1 次 → 新增 1 段 [1,2]

答案 = 3
```

```java
Map<Long, Integer> cnt = new HashMap<>();
cnt.put(0L, 1);
long pre = 0, ans = 0;
for (int x : a) {
    pre += x;
    ans += cnt.getOrDefault(pre - k, 0);
    cnt.merge(pre, 1, Integer::sum);
}
```

**图上理解**：前缀和像一串"路标"；两个路标差值是 k，就对应一段合法子数组。

## 7. 踩坑提醒 ⚠️

1. **数据会溢出**：元素 10⁹，n=10⁵，总和最大 10¹⁴ → **必须 long**。
2. **下标从 1 还是 0**：闭区间开区间别混，推荐**从 1 起，留 S[0]=0 哨兵**。
3. **差分 r+1 越界**：数组要开 `n+2`。
4. **区间减法别弄反**：`S[r] - S[l-1]`，不是 `S[r] - S[l]`。
