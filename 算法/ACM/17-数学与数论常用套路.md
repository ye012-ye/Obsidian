# 17 · 数学与数论常用套路

> 很多校赛题表面像思维题，实则是数论模板题。这里补一组最常见的工具。

---

## 1. gcd / lcm

### 1.1 最大公约数

```java
long gcd(long a, long b) {
    return b == 0 ? a : gcd(b, a % b);
}
```

### 1.2 最小公倍数

```java
long lcm(long a, long b) {
    return a / gcd(a, b) * b;
}
```

**先除再乘**，防止中间溢出。

---

## 2. 快速幂

求 `a^b mod mod`，不能真的乘 b 次。

### 2.1 思想

```
a^13 = a^(1101)_2
     = a^8 * a^4 * a^1
```

每次平方、按位判断即可。

### 2.2 模板

```java
long qpow(long a, long b, long mod) {
    long ans = 1 % mod;
    a %= mod;
    while (b > 0) {
        if ((b & 1) == 1) ans = ans * a % mod;
        a = a * a % mod;
        b >>= 1;
    }
    return ans;
}
```

## 3. 模逆元

如果要做除法：

> `a / b mod p`

在模意义下不能直接除，要乘上 `b` 的逆元。

当 `p` 是质数且 `b % p != 0` 时：

```
b^(p-1) ≡ 1 (mod p)
b^(p-2) ≡ b^(-1) (mod p)
```

所以：

```java
long inv(long b, long mod) {
    return qpow(b, mod - 2, mod);
}
```

## 4. 筛法与质因数分解

### 4.1 线性筛 / 埃氏筛

当题目要求：
- 求 `1..N` 所有素数
- 统计素数个数
- 预处理最小质因子

就要想到筛法。

### 4.2 试除分解

```java
Map<Long, Integer> factor(long n) {
    Map<Long, Integer> map = new HashMap<>();
    for (long i = 2; i * i <= n; i++) {
        while (n % i == 0) {
            map.merge(i, 1, Integer::sum);
            n /= i;
        }
    }
    if (n > 1) map.merge(n, 1, Integer::sum);
    return map;
}
```

## 5. 组合数 `C(n, k)`

### 5.1 小数据

可以直接杨辉三角：

```java
C[i][0] = C[i][i] = 1
C[i][j] = C[i-1][j-1] + C[i-1][j]
```

### 5.2 大数据取模

常用预处理：
- `fac[i] = i! mod p`
- `ifac[i] = (i!)^{-1} mod p`

则：

```java
C(n, k) = fac[n] * ifac[k] % mod * ifac[n-k] % mod
```

## 6. 同余与取模常见性质

```
(a + b) % mod = ((a % mod) + (b % mod)) % mod
(a - b) % mod = ((a % mod) - (b % mod) + mod) % mod
(a * b) % mod = ((a % mod) * (b % mod)) % mod
```

减法时要记得 `+mod`，防止负数。

## 7. 什么时候想到数论

关键词：
- 最大公约数 / 最小公倍数
- 质数 / 因子 / 约数个数
- 取模、幂、逆元
- 组合数
- 同余、余数、循环节

---

## 8. 踩坑提醒 ⚠️

1. **组合数取模**时，除法要用逆元。
2. `a / gcd(a,b) * b` 比 `a * b / gcd` 更安全。
3. 快速幂里 `a*a` 可能爆 `int`，用 `long`。
4. 分解质因数时，循环结束后若 `n > 1` 还要补一个因子。
5. 题目如果模数**不是质数**，`b^(mod-2)` 不一定是逆元。
