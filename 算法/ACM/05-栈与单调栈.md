# 05 · 栈与单调栈

> 栈很简单，但"单调栈"是很多 O(n²) 暴力题变 O(n) 的利器。

---

## 1. 栈（Stack）基础

### 1.1 原理：后进先出（LIFO）

```
push 顺序: 1, 2, 3
          ┌─┐     ┌─┐
          │3│ ←┐  │ │
          ├─┤   │ ├─┤
          │2│   │ │3│ ← pop 拿到 3
          ├─┤   │ ├─┤
          │1│   │ │2│
          └─┘     └─┘
        push 后   pop 后
```

### 1.2 Java 中怎么用

**不要用 `java.util.Stack`（老，慢）**，用 `ArrayDeque`：

```java
ArrayDeque<Integer> stack = new ArrayDeque<>();
stack.push(x);           // 入栈
int top = stack.peek();  // 看栈顶（不弹）
int t   = stack.pop();   // 弹出
boolean empty = stack.isEmpty();
int size = stack.size();
```

### 1.3 经典应用：括号匹配

```java
boolean valid(String s) {
    ArrayDeque<Character> st = new ArrayDeque<>();
    for (char c : s.toCharArray()) {
        if (c == '(' || c == '[' || c == '{') {
            st.push(c);
        } else {
            if (st.isEmpty()) return false;
            char t = st.pop();
            if (c == ')' && t != '(') return false;
            if (c == ']' && t != '[') return false;
            if (c == '}' && t != '{') return false;
        }
    }
    return st.isEmpty();
}
```

图示：
```
s = "({[]})"

 (  push →  [ (  ]
 {  push →  [ ( { ]
 [  push →  [ ( { [ ]
 ]  pop  →  栈顶 '[' 匹配 ']' ✓，[ ( { ]
 }  pop  →  栈顶 '{' 匹配 '}' ✓，[ ( ]
 )  pop  →  栈顶 '(' 匹配 ')' ✓，[ ]

空栈 → valid ✓
```

## 2. 单调栈（Monotonic Stack）

### 2.1 要解决的问题模板

**经典题**：给数组 a，求每个元素右边"第一个比它大的元素"的位置（没有则 -1）。

**暴力**：O(n²)，对每个 i 往右扫。
**单调栈**：O(n)。

### 2.2 原理 & 图解

> **维护一个单调递减的栈**（存的是"还没找到比自己大的"的元素下标）。

```
a = [2, 1, 3, 1, 4]

i=0, a[0]=2: 栈空 push(0)     栈(值): [2]
i=1, a[1]=1: 栈顶 2 > 1，push(1)    栈: [2, 1]
i=2, a[2]=3: 栈顶 1 < 3 → 弹出 1，其答案=2(下标)
             栈顶 2 < 3 → 弹出 0，其答案=2(下标)
             push(2)                栈: [3]
i=3, a[3]=1: 栈顶 3 > 1，push(3)    栈: [3, 1]
i=4, a[4]=4: 栈顶 1 < 4 → 弹出 3，其答案=4(下标)
             栈顶 3 < 4 → 弹出 2，其答案=4(下标)
             push(4)                栈: [4]

剩栈底未弹的 4 → 答案 -1（右边没有更大）

最终 next[] = [2, 2, 4, 4, -1]
             i=0 的右边第一个大于它的在下标 2 (值3)
```

### 2.3 万能模板

```java
int[] nextGreater(int[] a) {
    int n = a.length;
    int[] ans = new int[n];
    Arrays.fill(ans, -1);
    ArrayDeque<Integer> stk = new ArrayDeque<>();   // 存下标
    for (int i = 0; i < n; i++) {
        while (!stk.isEmpty() && a[stk.peek()] < a[i]) {
            ans[stk.pop()] = i;
        }
        stk.push(i);
    }
    return ans;
}
```

**变种一览**：

| 需求 | while 条件 |
|------|------------|
| 右边第一个**大于** | `a[stk.peek()] < a[i]` |
| 右边第一个**大于等于** | `a[stk.peek()] <= a[i]` |
| 右边第一个**小于** | `a[stk.peek()] > a[i]` |
| 左边第一个**大于** | 倒序扫，栈变成存"右"那一侧 |

## 3. 单调栈经典应用

### 3.1 柱状图最大矩形（LC 84，极其经典）

```
   高度:  [2, 1, 5, 6, 2, 3]
   
   ┌┐
   ││            ┌┐
   ││      ┌┐    ││
   ││      ││    ││
   ││   ┌┐ ││    ││┌┐
   │├┐  ││ ││ ┌┐ ││││
   ████ ▓▓ ▓▓ ██ ████

答案：最大矩形 = 10（5 和 6 组成宽 2 高 5 的矩形）
```

**思路**：对每个柱子 i，找它左右"第一个比它矮的"位置 l, r；
则以 i 为高的矩形宽度 = r - l - 1，面积 = a[i] * (r-l-1)。
两次单调栈即可 O(n)。

### 3.2 每日温度（LC 739）

题面：给每日温度 a，求每天"几天后才能遇到更高温度"。
直接套"右边第一个大于"模板。

## 4. 判断"何时想到单调栈"

关键词：
- "下一个更大 / 更小"
- "左边第一个更大 / 更小"
- "直方图的最大矩形"
- "区间内最左边比 x 小的"

**共同特征**：对每个元素找 "**某侧第一个满足某关系的元素**"。

---

## 5. 扩展进阶：贡献法看单调栈

单调栈不只找"右边第一个更大/更小"，还常用来算：
**每个元素作为最小值/最大值时，对总答案贡献多少次。**

### 例：所有子数组最小值之和（LC 907）

对每个 `a[i]`：
- 找左边第一个 **严格小于** 它的位置 `L`
- 找右边第一个 **小于等于** 它的位置 `R`

那么以 `a[i]` 作为最小值的子数组个数就是：
`(i - L) * (R - i)`

```
        L        i        R
... [更小]  x  x  a[i]  x  x [<=它]
           └左选法┘      └右选法┘

左边可选 i-L 种起点，右边可选 R-i 种终点
乘起来就是 a[i] 被当作最小值的次数
```

总贡献：
`ans += a[i] * (i - L) * (R - i)`

这类题属于单调栈的"进阶形态"：
先用单调栈找边界，再把边界变成贡献次数。

## 6. 踩坑提醒 ⚠️

1. **栈里存下标还是值？** 下标更通用（能查原值也能算距离）。
2. **别用 `Stack` 类**：`ArrayDeque` 快 5 倍。
3. **`peek` / `pop` 空栈报错**：一定先 `isEmpty()` 判断。
4. **严格大于 vs 大于等于**：根据题意选 `<` 还是 `<=`，结果可能不同。
5. **循环结束后栈里剩下的元素**：表示"右边没有更大的"，常常要处理。
