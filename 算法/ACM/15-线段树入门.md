# 15 · 线段树入门

> 如果树状数组是"前缀和工具"，那线段树就是"区间问题总控台"。

---

## 1. 它解决什么问题

数组上经常会遇到：
- 区间求和
- 区间最大值 / 最小值
- 单点修改
- 区间修改

暴力每次扫区间是 `O(n)`，会超时。

线段树可以把很多操作压到 `O(log n)`。

---

## 2. 结构怎么理解

把整个区间不断二分：

```
            [1,8]
          /       \
       [1,4]     [5,8]
      /   \      /   \
   [1,2] [3,4] [5,6] [7,8]
   ...
```

每个节点维护一个区间的信息，比如：
- 区间和
- 区间最大值
- 区间最小值

---

## 3. 为什么数组开 4n 就够

线段树虽然是树，但通常直接用数组存：

```java
long[] sum = new long[4 * n];
```

因为完全够放下所有节点，比赛里直接背这个结论即可。

---

## 4. 最基础：单点修改 + 区间求和

### 4.1 建树

```java
void build(int p, int l, int r, int[] a) {
    if (l == r) {
        sum[p] = a[l];
        return;
    }
    int mid = (l + r) >> 1;
    build(p << 1, l, mid, a);
    build(p << 1 | 1, mid + 1, r, a);
    sum[p] = sum[p << 1] + sum[p << 1 | 1];
}
```

### 4.2 单点修改

```java
void update(int p, int l, int r, int idx, int v) {
    if (l == r) {
        sum[p] = v;
        return;
    }
    int mid = (l + r) >> 1;
    if (idx <= mid) update(p << 1, l, mid, idx, v);
    else update(p << 1 | 1, mid + 1, r, idx, v);
    sum[p] = sum[p << 1] + sum[p << 1 | 1];
}
```

### 4.3 区间查询

```java
long query(int p, int l, int r, int ql, int qr) {
    if (ql <= l && r <= qr) return sum[p];
    int mid = (l + r) >> 1;
    long ans = 0;
    if (ql <= mid) ans += query(p << 1, l, mid, ql, qr);
    if (qr > mid) ans += query(p << 1 | 1, mid + 1, r, ql, qr);
    return ans;
}
```

## 5. 图解：查询区间 [3,7]

```
            [1,8]
          /       \
       [1,4]     [5,8]
      /   \      /   \
   [1,2] [3,4] [5,6] [7,8]
                 ...    ...

查 [3,7] 时：
- [1,2] 完全无关，跳过
- [3,4] 完全包含，拿来
- [5,6] 完全包含，拿来
- [7,8] 部分相交，继续往下拆
```

所以线段树查询不是扫整段，而是只访问少量相关节点。

---

## 6. 懒标记（Lazy Tag）

如果题目变成：

> 区间 `[l,r]` 每个数都加 v

那不能每次真的递归到叶子，不然会退化。

做法：
- 当前节点整个区间都被覆盖时，先把变化记在这个节点
- 以后真的访问子节点时，再把标记下传

这就叫**懒标记**。

### 区间加的关键思路

```java
sum[p] += (r - l + 1) * v;
lazy[p] += v;
```

图上理解：
```
给整个 [1,8] +3

不用立刻把 8 个叶子都改掉
只需记：
[1,8] 这个节点的 sum 已经 +24
并且 lazy=3

等以后有人查 [1,4]、[5,8]，再往下发
```

---

## 7. 什么时候用线段树

关键词：
- 区间查询 + 修改
- 动态维护区间最值
- 区间加、区间求和
- 在线处理，不能离线排序

如果只是前缀和类，优先树状数组；
如果是更复杂的区间信息，线段树更稳。

---

## 8. 踩坑提醒 ⚠️

1. **区间是闭区间还是半开区间**，必须统一。
2. **数组一般开 4n**，别省。
3. **中点写法**：`int mid = (l + r) >> 1;`
4. **pushUp / pushDown 别漏**，尤其懒标记题。
5. **区间和要用 long**。
