**1. NGINX 是什么？**  
NGINX是 Igor Sysoev 于 2002 年为解决 C10k（同时处理 10,000 个连接）问题而开发的高性能、异步事件驱动型 Web 服务器，2004 年首次发布。它以轻量、低资源、高并发闻名，成为当今最常用的 Web 服务器之一。

**2. 核心功能与用途**

- **静态 Web 服务**：高效响应 HTML、CSS、JS、图像等静态文件请求，性能优于传统 Apache。
- **反向代理 & 负载均衡**：将请求转发给后端群集，支持健康检测、会话保持等功能，帮助提升服务可用性和弹性。
- **HTTP 缓存**：作为前置缓存代理缓存后端响应，降低后端压力，加快响应速度。
- **SSL/TLS 终端**：处理 HTTPS 加密/解密，保护后端服务器资源。
- **Mail、TCP/UDP 代理**：支持 IMAP、POP3、SMTP 协议，以及流量代理转发能力。
