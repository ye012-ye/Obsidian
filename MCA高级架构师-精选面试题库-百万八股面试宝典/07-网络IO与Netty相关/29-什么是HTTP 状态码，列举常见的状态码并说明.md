HTTP 状态码是服务器在响应 HTTP 请求时返回的三位数字代码，用于指示请求的处理结果，帮助客户端了解接下来应该如何处理。M

常见的HTTP状态码如下：

- **200 OK**  
  表示请求已成功处理，服务器返回所请求的资源。常见于页面或 API 请求成功时。
- **301 Moved Permanently**  
  表示资源已永久移动到新 URL，客户端应使用返回的 Location 重定向到新地址，这对搜索引擎收录和 SEO 非常有用。S
- **400 Bad Request**  
  表示请求有语法错误或格式不正确，服务器无法解析。比如 URL 拼写错误或请求参数不可用。
- **404 Not Found**  
  表示服务器找不到所请求的资源。页面不存在、URL 输入错误、资源已被删除都会返回该状态码。
- **500 Internal Server Error**  
  表示服务器在处理请求时遇到未知错误，是一种通用的服务器端错误提示。
