Elasticsearch 是一个基于 Apache Lucene 的 **分布式、实时搜索与分析引擎**。它将文档以 JSON 形式存储，并通过倒排索引实现 **毫秒级全文检索**，同时支持结构化查询、过滤和复杂聚合操作。M

作为 NoSQL 数据存储，Elasticsearch 架构灵活、具备水平扩展能力，能够处理海量结构化或非结构化数据。数据被分成多个 shard 保存于集群各节点，保证了**高可用性与并行处理能力**。

其核心功能包括：S

- **全文搜索**：支持模糊、短语、布尔和正则等多种查询类型，具备丰富的相关性排名算法（如 BM25）。
- **实时分析**：能对日志、事件或指标数据进行快速聚合和统计，常用于监控、BI 和流量分析。
- **分布式存储**：通过 sharding 和 replication 实现数据冗余并支持多节点容错。
- **RESTful API**：提供标准的 HTTP 接口，方便与任何语言的应用程序集成。

常见应用场景有：B

- **网站搜索和应用内搜索**：提供快速精准的内容检索；
- **日志和事件分析**：作为 ELK（Elastic Stack）核心组件，用于日志索引、检索和可视化；
- **监控与可视化**：配合 Kibana 和 Beats 构建监控仪表盘与告警系统；
- **BI 与安全分析**：支持大规模数据的实时查询与聚合，常用于安全事件监测与业务分析。
