MyBatis 是一个轻量级的 **Java 持久化框架**，它通过 XML 配置或注解，将 Java 方法映射到 SQL 语句，同时自动处理 JDBC 的样板代码，如参数绑定和结果映射，大大简化数据库层的开发。

MyBatis 不会像全 ORM 一样将 Java 对象和数据库表结构一一绑定，而是允许你**手写 SQL**，精确控制查询逻辑。这使得它在处理复杂 SQL、存储过程或性能敏感场景时更具优势，同时也更适合集成遗留数据库或非标准结构的系统。

在运行时，MyBatis 会读取全局配置和映射文件（或注解），构建 `SqlSessionFactory` 和 `SqlSession`，通过动态代理将接口方法调用转化为 SQL 执行，最后将查询结果映射为 POJO。

MyBatis 还支持高级映射功能，如存储过程调用、多表关联、动态 SQL、缓存机制（一级/二级缓存）等，同时与 Spring 等框架集成自然流畅。
