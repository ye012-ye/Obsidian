Scala中Class和Object区别如下：

|  |  |  |
| --- | --- | --- |
| **特性** | **Class** | **Object** |
| **定义方式** | class | object |
| **实例化** | 需要new创建实例，实例化时加载 | 单例对象，不可实例化。程序启动时加载一次 |
| **参数传递** | 支持主构造器和辅助构造器 | 不支持，但可以通过apply实现类似传参 |
| **属性与方法** | 属于实例对象 | 属于静态，全局共享 |
| **生命周期** | 每实例化一次创建一份 | 单例，全局唯一 |
| **使用场景** | 描述对象结构和行为 | 工具类或常量的存放 |
