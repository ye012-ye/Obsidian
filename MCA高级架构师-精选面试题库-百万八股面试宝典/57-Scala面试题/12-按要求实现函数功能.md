给Scala List[Int]对象增加myForeach方法，可以直接使用List[Int]调用，该方法给List中每个元素增加100。

该需求可以通过隐式转换函数实现，代码如下：

|  |
| --- |
| class xx(list:List[Int]) {  def myForeach() ={  list.foreach(one=>{  *println*(one+100)  })  } } object ScalaTest{  *//隐式转换函数给List类型增加功能* implicit def myfun(list:List[Int]):xx={  new xx(list)  }   def main(args: Array[String]): Unit = {  val list = List[Int](1,2,3,4,5)  list.myForeach()*//结果：101、102、103、104、105* }  } |

以上代码中通过隐式转换函数 myfun，List[Int] 类型的对象在作用域内可以调用 xx类中定义的 myForeach 方法，而无需修改 List 类本身。
