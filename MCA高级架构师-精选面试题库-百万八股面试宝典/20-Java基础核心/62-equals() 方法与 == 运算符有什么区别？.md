在 Java 中，`==` 与 `equals()` 两者都可用于比较，但功能和场景截然不同：M

#### 1. 比较目标

- `==` 比较的是原始类型的值，或引用类型的**内存地址是否相同**。
- `equals()` 是 `Object` 类的方法，默认行为也比较地址，但大多数类（如 `String`, `Integer`, 自定义类）会重写它，以比较对象的**内容或状态是否相等**。

#### 2. 使用场景

- **原始类型**（如 `int`, `char`）只能用 `==`，适合快速比较值。S
- **对象类型**通常应使用 `equals()`，特别是内涵语义比较，如两个 `String` 对象内容不同但值相同，用 `equals()` 判定为相等，使用 `==` 通常为 `false`（除非它们指向同一个字符串常量）。B
