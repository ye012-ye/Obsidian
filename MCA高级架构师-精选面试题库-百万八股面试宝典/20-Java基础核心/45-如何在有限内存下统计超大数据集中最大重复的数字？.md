在有限内存的环境下处理超大数据集并统计数字的重复次数，可以使用几种优化方法。直接使用 `HashMap` 统计整个数据集容易导致内存溢出（OOM）。因此，我们可以采用如下策略：M

1. **分块读取数据：**  
   通过将数据分成若干块进行逐块处理，能够有效减少每次内存使用的峰值。每个块在内存中保持一个合适的大小，以便在内存限制内完成统计。
2. **使用滑动窗口：**  
   滑动窗口技术允许我们动态地调整处理的数据范围。每次处理一个窗口内的数据，通过统计数字的频率来找到窗口内最频繁出现的数字。
3. **稀疏和密集模式处理：**

- **稀疏模式：** 当窗口中的数字较为稀疏时，我们可以使用 `HashMap` 来记录每个数字的频率。
- **密集模式：** 当数据较为密集时，可以使用一个定长的数组来记录数字的频率。S

4. **临时文件存储：**  
   当哈希表的大小超过一定阈值时，可以将频率统计结果写入临时文件，以节省内存空间。
5. **动态窗口大小计算：**  
   根据内存限制和数据的最大值动态计算窗口的大小，以确保每个窗口在内存限制内。

**代码示例(分块读取和动态滑动窗口算法)：**

```java
package com.example;

import java.io.*;
import java.util.*;

public class FrequencyCounter {

    private static final long MAX_VALUE = 1_000_000_000; // 数字最大值
    private static final int MEMORY_LIMIT_MB = 100;     // 内存限制，单位MB
    private static final int INT_SIZE_BYTES = 4;        // 每个整数占用字节数
    private static final int THRESHOLD = 5_000_000;     // HashMap阈值

    public static void main(String[] args) throws IOException {
        String filePath = "data.txt"; // 输入文件路径

        int windowSize = calculateWindowSize(MEMORY_LIMIT_MB, MAX_VALUE, INT_SIZE_BYTES);
        long maxNum = -1;
        int maxFreq = -1;

        for (long start = 0; start < MAX_VALUE; start += windowSize) {
            long end = Math.min(start + windowSize, MAX_VALUE);
            boolean isSparse = isSparseWindow(start, end, filePath);

            Map<Long, Integer> sparseMap = null;
            int[] denseArray = null;

            if (isSparse) {
                sparseMap = new HashMap<>();
            } else {
                denseArray = new int[(int) (end - start)];
            }

            try (BufferedReader reader = new BufferedReader(new FileReader(filePath))) {
                String line;
                while ((line = reader.readLine()) != null) {
                    long number = Long.parseLong(line.trim());
                    if (number >= start && number < end) {
                        if (isSparse) {
                            sparseMap.put(number, sparseMap.getOrDefault(number, 0) + 1);
                            if (sparseMap.size() > THRESHOLD) {
                                dumpToDisk(sparseMap, start, end);
                                sparseMap.clear();
                            }
                        } else {
                            denseArray[(int) (number - start)]++;
                        }
                    }
                }
            }

            long[] localMax = findMaxFrequency(sparseMap, denseArray, start, end);
            if (localMax[1] > maxFreq) {
                maxFreq = (int) localMax[1];
                maxNum = localMax[0];
            }
        }

        System.out.println("最大重复次数的数字是: " + maxNum);
        System.out.println("重复次数: " + maxFreq);
    }

    private static int calculateWindowSize(int memoryLimitMB, long maxValue, int intSizeBytes) {
        int maxWindowSize = (memoryLimitMB * 1024 * 1024) / intSizeBytes;
        return (int) Math.max(1, Math.min(maxWindowSize, maxValue));
    }

    private static boolean isSparseWindow(long start, long end, String filePath) throws IOException {
        int sampleSize = 1000;
        int matchCount = 0;

        try (BufferedReader reader = new BufferedReader(new FileReader(filePath))) {
            String line;
            int sampleCount = 0;
            while ((line = reader.readLine()) != null && sampleCount < sampleSize) {
                long number = Long.parseLong(line.trim());
                if (number >= start && number < end) {
                    matchCount++;
                }
                sampleCount++;
            }
        }

        return (double) matchCount / sampleSize < 0.01;
    }

    private static void dumpToDisk(Map<Long, Integer> data, long start, long end) throws IOException {
        String fileName = "window_" + start + "_" + end + ".tmp";
        try (BufferedWriter writer = new BufferedWriter(new FileWriter(fileName, true))) {
            for (Map.Entry<Long, Integer> entry : data.entrySet()) {
                writer.write(entry.getKey() + "," + entry.getValue() + "\n");
            }
        }
        data.clear();
    }

    private static long[] findMaxFrequency(Map<Long, Integer> sparseMap, int[] denseArray, long start, long end) throws IOException {
        long maxNum = -1;
        int maxFreq = -1;

        if (denseArray != null) {
            for (int i = 0; i < denseArray.length; i++) {
                if (denseArray[i] > maxFreq) {
                    maxFreq = denseArray[i];
                    maxNum = start + i;
                }
            }
        }

        if (sparseMap != null) {
            for (Map.Entry<Long, Integer> entry : sparseMap.entrySet()) {
                if (entry.getValue() > maxFreq) {
                    maxFreq = entry.getValue();
                    maxNum = entry.getKey();
                }
            }
        }

        String fileName = "window_" + start + "_" + end + ".tmp";
        File tempFile = new File(fileName);
        if (tempFile.exists()) {
            try (BufferedReader reader = new BufferedReader(new FileReader(tempFile))) {
                String line;
                while ((line = reader.readLine()) != null) {
                    String[] parts = line.split(",");
                    long number = Long.parseLong(parts[0]);
                    int freq = Integer.parseInt(parts[1]);
                    if (freq > maxFreq) {
                        maxFreq = freq;
                        maxNum = number;
                    }
                }
            }
            tempFile.delete();
        }

        return new long[]{maxNum, maxFreq};
    }
}
```
