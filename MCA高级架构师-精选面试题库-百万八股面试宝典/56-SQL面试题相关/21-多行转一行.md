有如下表及数据，按照需求转换数据。

|  |
| --- |
| **#建表语句及插入数据**  CREATE TABLE people (  name STRING,  category STRING,  price INT  ) row format delimited fields terminated by '\t';  INSERT INTO people VALUES  ('zhangsan', 'A', 1),  ('zhangsan', 'B', 2),  ('zhangsan', 'C', 3),  ('lisi', 'A', 4),  ('lisi', 'C', 5),  ('zhangsan', 'D', 6),  ('lisi', 'B', 7),  ('wangwu', 'C', 8);  **#查询数据**  +--------------+------------------+---------------+  | people.name | people.category | people.price |  +--------------+------------------+---------------+  | zhangsan | A | 1 |  | zhangsan | B | 2 |  | zhangsan | C | 3 |  | lisi | A | 4 |  | lisi | C | 5 |  | zhangsan | D | 6 |  | lisi | B | 7 |  | wangwu | C | 8 |  +--------------+------------------+---------------+ |

将以上数据转换成如下格式：

|  |
| --- |
| +-----------+----------+--------------+  | name | str | total\_price |  +-----------+----------+--------------+  | lisi | A,C,B | 16 |  | wangwu | C | 8 |  | zhangsan | A,B,C,D | 12 |  +-----------+----------+--------------+ |

实现以上需求SQL语句如下：

|  |
| --- |
| #sql  select  name,concat\_ws(",",collect\_list(category)) as str,sum(price) as total\_price  from people  group by name |
