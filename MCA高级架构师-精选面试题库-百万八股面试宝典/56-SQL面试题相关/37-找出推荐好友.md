数据表friendship数据如下:

|  |  |
| --- | --- |
| **Col1** | **col2** |
| A | B |
| B | A |
| A | C |
| C | A |
| A | D |
| D | C |
| A | F |
| F | A |
| B | W |
| W | B |

在friendship表中SQL实现好友推荐：如果用户A和用户B互关，并且用户A和用户C互关，那么将B好友推荐给C或者C好友推荐给B，只需要找出这种好友推荐关系即可。

**SQL实现：**

|  |
| --- |
| SELECT o.recommend\_friend  FROM  (SELECT t.col2,  t.col1,  case  WHEN t.col2 < t.col3 THEN  concat(t.col2,t.col3)  ELSE concat(t.col3,t.col2)  END AS recommend\_friend  FROM  (SELECT a.col1,  a.col2,  b.col1 AS col3,  b.col2 AS col4  FROM friendship a  JOIN friendship b  ON a.col1 = b.col2  WHERE a.col2 !=b.col1) t ) o  GROUP BY o.recommend\_friend  HAVING count(o.recommend\_friend) >1 |
