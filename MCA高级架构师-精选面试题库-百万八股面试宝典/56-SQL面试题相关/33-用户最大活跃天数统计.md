| **#sql**  WITH temp1 AS (  SELECT  user\_id,  from\_unixtime(unix\_timestamp(activity\_dt,"yyyyMMdd"),"yyyy-MM-dd") as dt,  ROW\_NUMBER() OVER (PARTITION BY user\_id ORDER BY activity\_dt) AS rk,  date\_sub(from\_unixtime(unix\_timestamp(activity\_dt,"yyyyMMdd"),"yyyy-MM-dd"),ROW\_NUMBER() OVER (PARTITION BY user\_id ORDER BY activity\_dt)) as flag  FROM user\_activity  ),  temp2 AS (  SELECT  user\_id,  flag,  COUNT(\*) AS cnt  FROM temp1  GROUP BY user\_id, flag  )  SELECT  user\_id,  MAX(cnt) AS max\_days  FROM temp2  GROUP BY user\_id;  **#结果**  +----------+-----------+  | user\_id | max\_days |  +----------+-----------+  | uid1 | 5 |  | uid2 | 5 |  | uid3 | 5 |  | uid4 | 3 |  | uid5 | 3 |  | uid6 | 2 |  | uid7 | 2 |  | uid8 | 1 |  | uid9 | 1 |  +----------+-----------+ |
