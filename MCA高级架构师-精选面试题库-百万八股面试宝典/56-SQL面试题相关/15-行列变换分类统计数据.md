Hive中有个一个info表，如下：

|  |  |
| --- | --- |
| **date** | **result** |
| 2020-05-09 | win |
| 2020-05-09 | win |
| 2020-05-09 | lose |
| 2020-05-09 | lose |
| 2020-05-10 | win |
| 2020-05-10 | lose |
| 2020-05-10 | lose |
| ... | ... |

如果要生成下列结果，该如何写sql?

|  |  |  |
| --- | --- | --- |
| date | win | lose |
| 2020-05-09 | 2 | 2 |
| 2020-05-10 | 1 | 2 |
| ... | ... | ... |

文件info.txt数据：

|  |
| --- |
| 2020-05-09 win  2020-05-09 win  2020-05-09 lose  2020-05-09 lose  2020-05-10 win  2020-05-10 lose  2020-05-10 lose |

SQL建表语句：

|  |
| --- |
| create table info (dt string,result string) row format delimited fields terminated by '\t'; |

SQL查询语句:

|  |
| --- |
| select d,sum(case when r = 'win' then c else 0 end ) as win,sum(case when r = 'lose' then c else 0 end ) as lose from (select `date` as d ,result as r,count(result) as c from info group by `date`,result) t group by d; |
