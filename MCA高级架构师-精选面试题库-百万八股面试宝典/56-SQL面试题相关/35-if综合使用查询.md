| **#使用last\_value来统计所有数据中所有商品销售额都超过1000元的省份**  with temp as (select  province,  last\_value(amount) over (partition by province order by amount desc rows between unbounded preceding and unbounded following) as flag  from sale\_data  )  select distinct province from temp where flag >1000;  **#结果**  +-----------+  | province |  +-----------+  | 四川 |  | 河南 |  +-----------+ |
