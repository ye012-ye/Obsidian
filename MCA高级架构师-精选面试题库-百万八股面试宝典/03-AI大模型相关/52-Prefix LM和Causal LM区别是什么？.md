###### **Causal LM（因果语言模型）**

- **结构**：严格单向，仅用左侧上下文生成下一个词。
- **训练目标**：自回归预测（如GPT系列）。
- **示例**：输入“人工智能是”，生成“未来科技的核心”。
- **应用场景**：文本生成、对话系统。

###### **Prefix LM（前缀语言模型）**

- **结构**：输入分为前缀（可双向注意力）和生成部分（单向）。
- **训练目标**：前缀部分编码上下文，生成部分自回归预测（如T5）。
- **示例**：输入“翻译成英文：今天天气很好 →”，模型双向处理前缀“翻译成英文：今天天气很好”，再生成“The weather is nice today.”
- **应用场景**：文本编辑、翻译、摘要。

###### **对比总结**

|  |  |  |
| --- | --- | --- |
| **维度** | **Causal LM** | **Prefix LM** |
| **注意力方向** | 严格单向 | 前缀双向，生成单向 |
| **训练数据** | 自回归文本 | 带前缀的任务数据（如翻译） |
| **典型模型** | GPT、LLaMA | T5、UL2 |
