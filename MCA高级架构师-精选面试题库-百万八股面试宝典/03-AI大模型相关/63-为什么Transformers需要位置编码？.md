###### **核心原因**

自注意力机制本身是 **顺序无关的**。例如，句子 “A 打 B” 和 “B 打 A” 在无位置编码时，模型无法区分两者的差异，导致语义混乱。

###### **解决方案**

1. **绝对位置编码**：

- **实现**：为每个位置分配唯一向量，如 BERT 使用可学习的嵌入向量，位置 1 对应向量 ( E\_1 )，位置 2 对应 ( E\_2 )。
- **优点**：简单直观，适合短序列。
- **缺点**：无法外推到训练时未见的超长序列。

2. **相对位置编码**：

- **实现**：关注词之间的相对距离。例如，旋转位置编码（RoPE）将词向量按位置旋转，使注意力分数包含相对位置信息。
- **示例**：词 ( w\_i ) 和 ( w\_j ) 的注意力分数与 ( |i-j| ) 相关。
- **优点**：支持长文本外推，适合 LLaMA 等模型。

###### **实际影响**

- **短文本任务**（如分类）：绝对编码足够。
- **长文本生成**（如文档生成）：相对编码更优。
