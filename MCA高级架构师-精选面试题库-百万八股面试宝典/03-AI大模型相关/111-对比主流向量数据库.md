## 一、主流向量数据库简介

### **Milvus**

Milvus 是一个开源、分布式、企业级向量数据库，由 Zilliz 维护，专为 PB 级嵌入数据设计。其支持 GPU 加速、分片扩展、混合索引模式（如 HNSW、IVF、PQ 等），适合处理高并发大规模向量检索任务。

### **Qdrant**

Qdrant 是一个 Rust 实现的向量数据库，支持文本与图像多模态输入，适合私有化部署，具备高检索精度和安全性控制能力。对医疗、金融等敏感数据场景尤为适配。

### **Chroma**

ChromaDB 是为 LLM 应用设计的新兴开源向量数据库，注重开发体验和快速集成。它易部署、接口简洁，非常适合中小规模 RAG 原型验证或内部知识库使用。

### **FAISS**

FAISS 是 Meta 研发的高性能向量索引库，而非完整数据库，提供丰富索引结构（如 HNSW、IVF、PQ 等）和 GPU 加速支持，常用于离线批处理、学术实验场景。

### **Pinecone / Weaviate / pgVector**

- **Pinecone** 是一个托管的商用向量数据库 SaaS，适配大规模在线检索场景，用户无需管理底层部署。
- **Weaviate** 是一个开源多模态数据库，并带知识图谱支持，适合语义关联检索。
- **pgVector** 是 PostgreSQL 的扩展，适合向量检索需结合关系型存储的场景。

## 二、特点与适用场景

- **Milvus**：适合大规模、高并发、多节点部署场景，如电商推荐系统、内容召回系统。它支持 GPU/PQ、多向量查询、混合搜索等功能，扩展能力强。
- **Qdrant**：适合多模态任务与私有化部署需求，检索精度高，安全合规要求强的领域（如医学影像＋文本检索）非常适配。
- **Chroma**：轻量级、快速集成、LangChain 生态兼容好，适用于中小规模知识检索、内部助手、原型开发等场景。
- **FAISS**：适合科研实验或离线聚类、批量相似性搜索任务，但不提供数据库层面的运维功能。
- **Pinecone / Weaviate / pgVector**：

- Pinecone 适合企业级 SaaS 使用场景，无需自行构建集群。
- Weaviate 适合结合知识图谱查询语义丰富任务；
- pgVector 适合希望与 SQL 数据库融合使用的系统。
