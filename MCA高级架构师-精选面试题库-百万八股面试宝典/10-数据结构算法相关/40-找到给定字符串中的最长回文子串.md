核心思路：通过“两种中心”向两侧扩展，每次比较并更新当前最长回文子串的起始和结束位置。时间复杂度 O(n²)，空间 O(1)。

#### 完整 Java 代码如下：

```java
public class LongestPalindromicSubstring {

    // 用中心扩展法求最长回文子串
    public static String longestPalindrome(String s) {
        if (s == null || s.length() < 1) {
            return "";
        }
        int start = 0, end = 0;
        for (int i = 0; i < s.length(); i++) {
            // 单字符中心
            int len1 = expandAroundCenter(s, i, i); // M
            // 双字符中心
            int len2 = expandAroundCenter(s, i, i + 1); // S
            int len = Math.max(len1, len2);
            if (len > end - start) { // B
                start = i - (len - 1) / 2;
                end = i + len / 2;
            }
        }
        return s.substring(start, end + 1);
    }

    // 从两个指针作为中心扩展，返回回文长度
    private static int expandAroundCenter(String s, int left, int right) {
        while (left >= 0 && right < s.length()
               && s.charAt(left) == s.charAt(right)) {
            left--;
            right++;
        }
        return right - left - 1;
    }

    // 测试入口
    public static void main(String[] args) {
        String[] tests = {
            "babad",    // 期望 "bab" 或 "aba"
            "cbbd",     // 期望 "bb"
            "forgeeksskeegfor" // 期望 "geeksskeeg"
        };

        for (String s : tests) {
            String res = longestPalindrome(s);
            System.out.printf(
                "Input: \"%s\" → Longest Palindrome: \"%s\"%n", 
                s, res);
        }
    }
}
```

### 思路分析

1. 对每个位置 `i`，分别以 `(i,i)` 和 `(i,i+1)` 作为中心，判断奇偶长度的回文。
2. `expandAroundCenter` 方法向两边延伸，直到不满足回文条件后停止，并计算当前长度。
3. 使用全局变量 `start` 和 `end` 记录最长回文子串的左右边界。
4. 最终通过 `substring` 提取结果。

### 复杂度分析

- **时间复杂度**：O(n²)，主循环 O(n)，每次扩展最坏 O(n)。
- ​**空间复杂度**：O(1)，仅额外使用少量常数级变量。
