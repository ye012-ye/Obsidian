需求： 实现一个字符串匹配函数 `naiveSearch(text, pattern)`，查找并返回首次匹配位置，未找到则返回 -1。

### 代码实现示例

```java
public static int naiveSearch(String text, String pattern) {
    int n = text.length(), m = pattern.length();
    for (int i = 0; i <= n - m; i++) {
        int j = 0;
        while (j < m && text.charAt(i + j) == pattern.charAt(j)) {
            j++;
        }
        if (j == m) return i;
    }
    return -1;
}
```

该方法从 `text` 中每个可能的起始位开始字符逐一比对，如所有字符匹配则立即返回位置，否则继续滑动窗口。
