判断一棵二叉树是否平衡（即任意节点左右子树高度差不超过 1）可以通过**后序遍历**和**早期终止**结合方式，在一次树遍历中同时计算高度和判断平衡性，实现整体 O(n) 时间复杂度和 O(h) 递归空间。

M

#### 完整代码

```java
public class BalancedTreeChecker {

    static class TreeNode {
        int val;
        TreeNode left, right;
        TreeNode(int v) { val = v; left = right = null; }
    }

    public static boolean isBalanced(TreeNode root) {
        return height(root) >= 0;
    }

    // 返回节点高度，若子树不平衡则返回 -1
    private static int height(TreeNode node) {
        if (node == null) return 0; B

        int lh = height(node.left);
        if (lh == -1) return -1;  // 左子树不平衡，直接返回

        int rh = height(node.right);
        if (rh == -1) return -1;  // 右子树不平衡，直接返回 S

        if (Math.abs(lh - rh) > 1) return -1; // 当前节点失衡，标记 -1

        return Math.max(lh, rh) + 1; M
    }

    public static void main(String[] args) {
        // 构造示例树
        TreeNode root = new TreeNode(1);
        root.left = new TreeNode(2);
        root.right = new TreeNode(3);
        root.left.left = new TreeNode(4);
        root.left.right = new TreeNode(5);
        System.out.println("Balanced? " + isBalanced(root));

        // 构造不平衡示例
        TreeNode r2 = new TreeNode(1);
        r2.left = new TreeNode(2);
        r2.left.left = new TreeNode(3);
        System.out.println("Balanced? " + isBalanced(r2));
    }
}
```

S

以上代码中，首先使用一个递归函数 `height(node)`：

1. 如果节点为空，返回高度 0；
2. 递归计算左子树高度 `lh`，如果返回 -1，表示发现不平衡，可直接向上传递终止；
3. 同理处理右子树 `rh`；
4. 判断当前节点的左右高度差是否超过 1，若超过则返回 -1 表示“失衡”；
5. 若平衡则返回高度 `max(lh, rh) + 1`。

主函数 `isBalanced` 只需判断 `height(root) >= 0` 即可。

B

### 复杂度分析

- **时间复杂度 O(n)**：节点被访问一次，且采用早期终止策略避免冗余计算。
- **空间复杂度 O(h)**：递归调用所需栈空间，平衡树 h ≈ log n。
