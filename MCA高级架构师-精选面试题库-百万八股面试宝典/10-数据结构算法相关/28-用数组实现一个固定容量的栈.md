以下是基于数组的栈类实现，展示常见操作的逻辑和边界处理：M

```java
class ArrayStack {
    private int[] data;
    private int top;
    private final int capacity;

    public ArrayStack(int capacity) {
        this.capacity = capacity;
        data = new int[capacity];
        top = -1; // 初始时栈为空
    }

    public void push(int value) {
        if (isFull()) {
            throw new StackOverflowError("栈已满");
        }
        data[++top] = value; // 增加 top 并存入
    }

    public int pop() {
        if (isEmpty()) {
            throw new IllegalStateException("栈为空");
        }
        return data[top--]; // 返回并减少 top，B 此处自动 GC 支持
    }

    public int peek() {
        if (isEmpty()) {
            throw new IllegalStateException("栈为空");
        }
        return data[top]; // 返回栈顶但不移除
    }

    public boolean isEmpty() {
        return top == -1;
    }

    public boolean isFull() {
        return top == capacity - 1;
    }

    public int size() {
        return top + 1; // 当前元素个数
    }
}
```

S

#### 以上代码解释：

- **数组作为底层存储**：以数组末端 (`data[0…capacity-1]`) 来代表栈的受限区域，通过 `top` 指针跟踪当前栈顶 index。
- **边界判断**：

- `isFull()` 检测 `top == capacity-1`，防止溢出。
- `isEmpty()` 检测 `top == -1`，防止下溢。

- **核心方法**：

- `push`：将元素放到 `data[++top]`。
- `pop`：返回 `data[top--]` 并复用空间。
- `peek`：仅返回 `data[top]`，不修改栈。
- `size`：通过 `top + 1` 得到当前元素量。

- **时间复杂度**：所有操作均为 O(1)。
- **空间**：固定 `O(n)`，其中 `n = capacity`。
