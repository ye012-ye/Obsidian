可采用 Floyd “龟兔赛跑”算法，使用两个指针——慢指针每次走一步，快指针每次走两步——在 O(n) 时间与 O(1) 空间内高效判断链表是否有环。

M

#### 原理概述

当链表存在环时，快指针终将追上慢指针；否则快指针会提前到达链表末端 (`null`)，这一方法不依赖额外空间。

#### Java 代码

```java

public class LinkedListCycleDetector {

    static class ListNode {
        int val;
        ListNode next;
        ListNode(int v) { val = v; next = null; }
    }

    // 检测链表是否有环
    public static boolean hasCycle(ListNode head) {
        if (head == null || head.next == null) return false;

        ListNode slow = head;
        ListNode fast = head;

        while (fast != null && fast.next != null) {
            slow = slow.next;            // 慢指针走一步
            fast = fast.next.next;       // 快指针走两步
            if (slow == fast) {
                return true;             // 相遇，则存在环
            }
        }

        return false;  // 遍历结束未相遇，无环
    }

    // 创建测试链表并检测环
    public static void main(String[] args) {
        // 构建链表：1->2->3->4->5
        ListNode head = new ListNode(1);
        head.next = new ListNode(2);
        head.next.next = new ListNode(3);
        head.next.next.next = new ListNode(4);
        head.next.next.next.next = new ListNode(5);

        // 测试无环情况
        System.out.println("No cycle? " + hasCycle(head));

        // 手动设立环：5 指向 3
        head.next.next.next.next.next = head.next.next;

        // 测试有环情况
        System.out.println("Has cycle? " + hasCycle(head));
    }
}
```

#### 复杂度分析

- **时间复杂度 O(n)**：每个节点最多被快慢指针访问一次。
- **空间复杂度 O(1)**：仅使用两个指针，无需额外存储结构。
