我们可以通过只使用O(1)额外空间、反复重连节点指针的方式实现链表的原地逆转。以下是思路与实现：

首先，定义三个指针：`prev`（初始化为null），`curr`（当前节点，初始指向head），以及辅助的`nextNode`。

M

理解这三个指针是关键，因为它们对应链表当前遍历的节点状态。

1. 在循环中，首先保存当前节点的下一个节点：`nextNode = curr.next;`，避免断链丢失余下部分。
2. 将当前节点的`next`指向前一个节点：`curr.next = prev;`，实现局部反转，这是整个逆转过程的核心。
3. 将`prev`和`curr`依次移动一位：`prev = curr; curr = nextNode;`，继续处理剩余所有节点。
4. 反复上述操作直至`curr`为null，此时`prev`恰为新链表的头。
5. 返回`prev`即完成链表整体逆转。

S

以下是完整Java实现，采用常见命名风格：

```java
class ListNode {
    int val;
    ListNode next;
    ListNode(int v) { val = v; next = null; }
}

public class Solution {
    public static ListNode reverseList(ListNode head) {
        ListNode prev = null;
        ListNode curr = head;
        while (curr != null) {
            ListNode nextNode = curr.next; 
            curr.next = prev;
            prev = curr;
            curr = nextNode;
        }
        return prev;
    }
}
```

**复杂度分析：**

- 时间复杂度是O(n)，因为每个节点遍历一次；
- 空间复杂度是O(1)，仅使用固定数量的指针变量；
