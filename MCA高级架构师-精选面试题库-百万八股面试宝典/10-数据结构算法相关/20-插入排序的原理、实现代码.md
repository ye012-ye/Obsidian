插入排序通过将数组分为“已排序”和“未排序”两部分实现排序。初始时仅第一个元素视为已排序，从第二个元素开始，每次将当前元素与已排序部分从右向左比较，找到合适位置后将其插入，并将比它大的元素依次后移。这个过程类似于人们整理手中扑克牌，直观且易于实现。M

下面是 Java 的朴素实现代码：

```java

void insertionSort(int[] arr) {
    for (int i = 1; i < arr.length; i++) {
        int key = arr[i];
        int j = i - 1;
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j--;
        }
        arr[j + 1] = key;
    }
}
```

以上代码展示了插入排序的核心操作：通过向左扫描和元素移动完成当前元素正确插入。S

虽然插入排序最坏和平均时间复杂度为 O(n²)，但当数据量较小或输入数组近乎有序时，它的性能接近 O(n)，甚至优于一些复杂排序（如快速排序）在“小数组”或“接近有序”场景下的表现。同时，该算法是**稳定的**（不会改变相等元素间相对顺序）、**原地排序**（空间复杂度为 O(1)），适合在线排序或作为复杂排序的基础模块。B
