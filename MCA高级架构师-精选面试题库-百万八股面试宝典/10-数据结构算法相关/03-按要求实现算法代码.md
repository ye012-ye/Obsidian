需求： 在一个有序数组 `nums` 中查找目标值 `target`，如果存在则返回索引，否则返回它应该插入的位置。请简要说明思路并给出完整代码。

​

我们可以利用二分查找算法在有序数组中高效定位目标或者确定其插入位置。该方法通过每次将查找区间减半，时间复杂度稳定为 **O(log n)**。

在实现时，会初始化 `left=0` 与 `right=nums.length-1`。在循环中，使用 `mid = left + (right - left) / 2` 防止整数溢出。若 `nums[mid]` 等于目标，则直接返回 `mid`；若小于目标，说明插入位置或目标在右半部分，将 `left = mid + 1`；若大于目标，继续调整 `right = mid - 1`。循环结束时，`left` 即为目标应插入的位置或最终索引位置。

下面是 Java 版本的完整实现：

```java
public class SearchInsertPosition {
    public static int searchInsert(int[] nums, int target) {
        int left = 0, right = nums.length - 1;
        while (left <= right) {
            int mid = left + (right - left) / 2;
            if (nums[mid] == target) {
                return mid;
            } else if (nums[mid] < target) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }
        // 循环结束时 left 扩展到插入位置
        return left;
    }

    public static void main(String[] args) {
        int[] nums = {1, 3, 5, 6};
        System.out.println(searchInsert(nums, 5)); // 2
        System.out.println(searchInsert(nums, 2)); // 1
        System.out.println(searchInsert(nums, 7)); // 4
        System.out.println(searchInsert(nums, 0)); // 0
    }
}
```
